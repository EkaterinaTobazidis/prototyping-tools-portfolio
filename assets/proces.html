Reflectie en Ontwikkeling

Hier staat de verder documentatie van mijn proces in de overige vorige weken

verdere aanpassingen

de vorige weken heb ik gewerkt rond de schermen in figma. Hier zat ik op het begin een beetje bij vast maar heb uiteindelijk een resultaat gekregen waar ik blij mee was. Voor het gedeelte met de code heb ik heel lang vast gezeten aangezien mijn bestand op een bepaald moment niet werkte. Ik heb toen heel lang gezocht naar wat er fout liep en had uiteindekijk de fout gevonden. Hierna ben ik dan eindelijk verder kunnen gaan met hoe alles eruit moest zien en wat ik nog kon aanpassen Wat ik nog aangepast heb in Figma waren de GSM schermen. Hier ben ik opzoeke gegaan naar een manier om mijn vorige schermen om te zetten naar een scherm voor op je gsm. Dat lukte wel vrij vlot. Na het uit te proberen moest ik nog een paar dingen verzetten en uiteindelijk had ik het resuktaat dat ik wou hebben.

wat heb ik geleerd

Heel het code gedeelte was nieuw voor mij, ik had hiervoor nog nooit van HTML of CSS gehoord, dus het voelde heel nieuw aan. Op het begin was ik mee en ook wel niuewsgierig maar uiteindelijk zat ik toch even vast. Ik heb toen geprobeerd om mijn fouten te vinden en er dan een oplossing voor te vinden. Uiteindelijk is het toch gelukt en ben ik verder kunnen gaan. IK heb zeker dingen bijgeleerd maar het was vooral zoeken naar hoe alles in elkaar zat. Het gedeelte op Figma was ook nieuw maar wel interessant om kennis mee te maken.
